# Cryptography Signal Workbench

A front-end cryptography lab for encoding/decoding text and inspecting statistical traits of ciphertext.

## Highlights
- Caesar and Vigenere encryption/decryption modes.
- Letter-frequency chart rendered on Canvas.
- Entropy and Index of Coincidence metrics.
- Heuristic classification of cipher family based on signal profile.

## Stack
- HTML/CSS/Vanilla JS
- Canvas chart rendering

## Run
Open `index.html` in a browser.
